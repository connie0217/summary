BOM

BOM：浏览器对象模型；

BOM是浏览器提供的相关方法；

窗口

 window.open(url,name,specs)

  name:打开方式，框架name

打开方式：

_blank:新窗口打开；

_self本窗口打开；

_top:最外层框架

_parent：父级框架

  specs:规格，只有在新窗口打开有效

只有在新窗口打开有效 ,'width=100px,height=100px',属性=值，多个属性间用逗号分隔

 window.close();

var opener = null;

//打开新窗口

btn.onclick = function(){

//记录新打开窗口的window

opener = open();

}

//关闭新窗口

btn1.onclick = function(){

//通过记录找到新窗口的window关闭

opener.close();

}

//关闭当前窗口

  火狐下只能关闭open打开的新窗口
